# Seer - Frontend

React SPA for Seer prediction market platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
REACT_APP_API_URL=http://localhost:3000
```

3. Start development server:
```bash
npm start
```

The app will run on `http://localhost:3001` (React default port).
