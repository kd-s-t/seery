# Next.js Frontend

Next.js 15 application for Seer prediction market platform.

## Quick Start

**1. Install dependencies:**
```bash
npm install
```

**2. Set up environment:**
```bash
echo "NEXT_PUBLIC_API_URL=http://localhost:3016" > .env.local
```

**3. Run development server:**
```bash
npm run dev
```

App runs on `http://localhost:3015`

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Material-UI v5 (MUI5)
- Framer Motion
- ethers.js 6.15.0
- MetaMask integration

See main README.md for full documentation.
